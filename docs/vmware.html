Broadcom 계정 생성 (필수, 무료)

VMware는 Broadcom이 인수한 후 개인용 무료로 전환됐습니다. 다운로드 전 Broadcom 계정이 반드시 필요합니다.

① 회원가입

  1. https://support.broadcom.com 접속
  2. 우측 상단 [Register] 클릭
  3. 이메일 주소 입력 → [Next]
  4. 이름, 비밀번호 입력 → 가입 완료
  5. 이메일 인증 링크 클릭 (메일함 확인)

② 프로필 완성 (중요!)

가입 후 프로필이 불완전하면 다운로드가 차단됩니다. 반드시 완성하세요.

  1. 로그인 후 우측 상단 프로필 아이콘 클릭
  2. [My Profile] → 프로필 편집 클릭
  3. 아래 항목 모두 입력:
    • First Name / Last Name
    • Country: South Korea
    • Address (대략적으로 입력해도 됨)
    • Phone Number
  4. 저장 후 다시 다운로드 시도
⚠️ 프로필 미완성 시 다운로드 버튼이 비활성화되거나 오류 발생! 주소·전화번호까지 입력해야 합니다.
✅ 계정 생성 완전 무료 — 카드 정보 불필요

VMware Workstation Pro 다운로드

⚠️ Broadcom 포털은 복잡합니다. 아래 순서를 그대로 따라하세요!

① 정확한 다운로드 경로

  1. 아래 직접 링크 클릭 (로그인 후 이동됨)
https://support.broadcom.com/group/ecx/productdownloads?subfamily=VMware+Workstation+Pro

링크 클릭 시 로그인 페이지로 이동 → 로그인 → 자동으로 다운로드 페이지로 이동

② 수동으로 찾아가는 경우

  1. 로그인 후 상단 메뉴 [My Downloads] 클릭
  2. 검색창에 VMware Workstation Pro 입력 후 검색
  3. 결과에서 VMware Workstation Pro 클릭
  4. 버전 목록에서 최신 버전 선택 (예: 17.x.x)
  5. [I agree to Terms and Conditions] 체크
  6. 파일 목록에서 VMware-workstation-full-XX.X.X-XXXXXXXX.exe 옆 ⬇️ 클릭

③ 다운로드 안 될 때 체크리스트

증상해결법
다운로드 버튼 비활성화프로필 완성 필요 (STEP 1 ② 참고)
로그인 후 다시 로그인 요구브라우저 쿠키 허용, 시크릿 모드 해제
제품이 검색 안 됨직접 링크로 접속 (위 URL 사용)
다운로드 속도 매우 느림정상 (해외 서버). VPN 사용 금지
💡 다운로드 파일 크기: 약 600~700MB

VMware Workstation Pro 설치

① 설치 파일 실행

  1. 다운로드한 .exe 파일 우클릭 → 관리자 권한으로 실행
  2. [Next] 클릭하여 진행

② 설치 옵션

항목권장 설정

  3. VMware 실행 후 "Use VMware Workstation for Personal Use" 선택
  4. 라이선스 키 없이 무료 사용 가능 ✅

Windows 11 ISO 다운로드

  1. https://www.microsoft.com/ko-kr/software-download/windows11 접속
  2. 페이지 중간 "Windows 11 디스크 이미지(ISO) 다운로드" 섹션으로 스크롤
  3. 드롭다운에서 Windows 11 (multi-edition ISO) 선택 → [다운로드]
  4. 언어: 한국어 → [확인]
  5. [64비트 다운로드] 클릭 → ISO 파일 저장
💡 ISO 파일 크기 약 5~6GB, 다운로드 10~30분 소요. 저장 위치 기억해두세요!

가상머신 생성

① 새 가상머신 만들기

  1. VMware 실행 → [Create a New Virtual Machine] 클릭
  2. "Typical (recommended)" 선택 → [Next]
  3. "Installer disc image file (iso)" 선택 → [Browse] → ISO 파일 선택

② 저장 위치 설정

이름: Windows 11 위치: D:\VMs\Windows11 (용량 큰 드라이브 권장)

③ 하드웨어 설정 (중요!)

[Customize Hardware] 클릭 후 설정:

항목권장값최소값
디스크 크기80GB 이상64GB
Memory (RAM)8GB4GB
ProcessorsCPU 코어의 절반2코어
Network AdapterNAT (기본값)-
⚠️ RAM 4GB 미만이면 Windows 11이 매우 느립니다. 최소 4GB, 권장 8GB!

Windows 11 설치

① 가상머신 시작

  1. 생성된 가상머신 선택 → [Play virtual machine]
  2. "Press any key to boot from CD..." 메시지 뜨면 즉시 아무 키 입력!
⚠️ 키 입력 타이밍을 놓치면 다시 재시작해야 합니다. 부팅 화면 뜨자마자 클릭!

② 설치 단계

  1. 언어: 한국어 → [다음] → [지금 설치]
  2. 제품 키: "제품 키가 없음" 클릭 (나중에 정품 인증 가능)
  3. 버전: Windows 11 Pro 권장 → [다음]
  4. 사용 조건 동의 → [다음]
  5. 설치 유형: "사용자 지정: Windows만 설치"
  6. 파티션 선택 → [다음]
✅ 설치 시간 약 15~30분. 재시작 여러 번 반복되는 것은 정상입니다!

VMware Tools 설치 (필수!)

설치 완료 후 VMware Tools를 반드시 설치해야 화면 해상도·복사/붙여넣기·파일 공유가 정상 작동합니다.

  1. VMware 상단 메뉴 → [VM] → [Install VMware Tools]
  2. 가상머신 탐색기에서 DVD 드라이브 열기
  3. setup64.exe 실행 → [Next] → [Install] → [Finish]
  4. 재시작
⚠️ VMware Tools 없으면: 화면 작음, 복붙 안됨, 파일 공유 불가 → 반드시 설치!

❓ 자주 묻는 질문

Q. Broadcom 계정 없이 다운로드 가능한가요?
아니요. 2024년부터 Broadcom 무료 계정 로그인이 필수입니다. 가입 완전 무료이며 카드 정보도 필요 없습니다. 단, 프로필(이름·주소·전화번호)을 완성해야 다운로드 버튼이 활성화됩니다.
Q. 다운로드 버튼이 회색으로 비활성화돼요
Broadcom 프로필이 불완전할 때 발생합니다. 로그인 후 My Profile → 주소, 전화번호, 국가 정보를 모두 입력하고 저장 후 다시 시도하세요.
Q. 로그인했는데 계속 로그인 페이지로 돌아와요
브라우저 쿠키 문제입니다. ① 시크릿 모드 해제 ② 쿠키 허용 설정 확인 ③ Chrome 또는 Edge 사용을 권장합니다. Safari는 차단될 수 있습니다.
Q. Windows 11이 TPM 오류로 설치 안 돼요
가상머신 설정 → [VM] → [Settings] → [Options] → [Advanced] → UEFI 선택 + Secure Boot 체크하세요.
Q. 가상머신이 너무 느려요
RAM을 8GB 이상으로 늘리고, VMware Tools가 설치됐는지 확인하세요. Tools 없으면 무조건 느립니다.
Q. 호스트 PC와 파일 공유하고 싶어요
VMware Tools 설치 후 → [VM] → [Settings] → [Shared Folders] → 공유할 폴더 추가하면 됩니다.
Q. 복사/붙여넣기가 안 돼요
VMware Tools가 설치되어 있어야 합니다. 설치 후 재시작하면 바로 됩니다.
