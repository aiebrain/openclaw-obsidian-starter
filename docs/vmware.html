Broadcom 계정 생성 (필수, 무료)

VMware는 Broadcom이 인수한 후 개인용 무료로 전환됐습니다. 다운로드 전 Broadcom 계정이 반드시 필요합니다.

① 회원가입

⚠️ 네이버·다음·네이트 이메일(@naver.com, @daum.net, @nate.com)로는 가입 불가!
반드시 Gmail(@gmail.com) 또는 회사 이메일 사용
  1. https://www.broadcom.com 접속
  2. 우측 상단 [Support Portal] → [Register] 클릭
  3. Gmail 주소 입력 후 진행
  4. 이메일로 제목: "Broadcom Registration Notice - Verification Code" 수신
  5. 본문의 6자리 숫자 인증 코드 입력
  6. 이름, 비밀번호 등 필수 정보 입력 → 이용약관 동의 → 가입 완료
  7. "Successfully!" 메시지 확인

② 프로필 설정

가입 후 프로필 생성 화면이 나타납니다.

💡 개인 사용자는 "I'll do it later" 클릭해도 됩니다. 프로필 없이도 다운로드 가능!

→ 이후 브로드컴 메인화면으로 자동 이동. 로그아웃됐다면 [Go To Portal] → 로그인

✅ 계정 생성 완전 무료 — 카드 정보 불필요

VMware Workstation Pro 다운로드

⚠️ Broadcom 포털은 복잡합니다. 아래 순서를 그대로 따라하세요!

① 정확한 다운로드 경로

  1. 아래 직접 링크 클릭 (로그인 후 이동됨)
https://support.broadcom.com/group/ecx/productdownloads?subfamily=VMware+Workstation+Pro

링크 클릭 시 로그인 페이지로 이동 → 로그인 → 자동으로 다운로드 페이지로 이동

② 정확한 메뉴 경로 (수동)

  1. 로그인 후 메인화면에서 [Software] 클릭
  2. [VMware Cloud Foundation] 선택
  3. [My Downloads] 클릭
  4. 목록 하단에서 [VMware Workstation Pro] 클릭
  5. 최신 버전 선택 (예: 17.x.x)
  6. "다운로드하려면 추가 인증이 필요합니다" 메시지 → 주소 입력 화면으로 이동
  7. 주소 입력 (형식상 대충 입력 가능, 우편번호만 정확히 입력)
  8. [I Agree] 동의 → 다시 다운로드 버튼 클릭
  9. VMware-workstation-full-XX.X.X-XXXXXXXX.exe ⬇️ 클릭
⚠️ 주소 입력 단계가 나타나는 건 정상! 귀찮아도 입력해야 무료 다운로드 가능합니다.

③ 다운로드 안 될 때 체크리스트

증상해결법
가입 자체가 안 됨Gmail 사용! 네이버·다음·네이트 이메일 불가
인증 코드 메일 안 옴스팸 폴더 확인 / 제목: "Broadcom Registration Notice"
다운로드 버튼 비활성화주소 추가 인증 단계 진행 (우편번호 입력)
메뉴에서 제품 못 찾음직접 링크 사용 (위 URL)
로그인 후 다시 로그인 요구Chrome/Edge 사용, 시크릿 모드 해제
다운로드 속도 느림정상 (해외 서버). VPN 사용 금지
💡 다운로드 파일 크기: 약 600~700MB

VMware Workstation Pro 설치

① 설치 파일 실행

  1. 다운로드한 .exe 파일 우클릭 → 관리자 권한으로 실행
  2. [Next] 클릭하여 진행

② 설치 옵션

항목권장 설정
설치 경로기본값 유지 (C:\Program Files\VMware\)
Enhanced Keyboard Driver✅ 체크 권장

Join the VMware CEIP체크 해제 가능

③ 설치 완료 후 첫 실행

  1. [Install] → 설치 진행 (약 3~5분)
  2. [Finish] → PC 재시작 권장
  3. VMware 실행 → 라이선스 키 입력 화면이 나오면 그냥 건너뛰기

④ 정상 설치 확인

VMware 실행 후 우측 상단 또는 About 화면에서 아래 상태 확인:

항목표시값의미
TypePersonal Use Only✅ 정상
StatusNot Licensed✅ 정상 (무료 버전)
💡 "Not Licensed"가 표시돼도 정상입니다! 2024년 11월부터 라이선스 없이 완전 무료 사용 가능.

Windows 11 ISO 다운로드

  1. https://www.microsoft.com/ko-kr/software-download/windows11 접속
  2. 페이지 중간 "Windows 11 디스크 이미지(ISO) 다운로드" 섹션으로 스크롤
  3. 드롭다운에서 Windows 11 (multi-edition ISO) 선택 → [다운로드]
  4. 언어: 한국어 → [확인]
  5. [64비트 다운로드] 클릭 → ISO 파일 저장
💡 ISO 파일 크기 약 5~6GB, 다운로드 10~30분 소요. 저장 위치 기억해두세요!

가상머신 생성

① 새 가상머신 만들기

  1. VMware 실행 → [Create a New Virtual Machine] 클릭
  2. "Typical (recommended)" 선택 → [Next]
  3. "Installer disc image file (iso)" 선택 → [Browse] → ISO 파일 선택

② 저장 위치 설정

이름: Windows 11 위치: D:\VMs\Windows11 (용량 큰 드라이브 권장)

③ 하드웨어 설정 (중요!)

[Customize Hardware] 클릭 후 설정:

항목권장값최소값
디스크 크기80GB 이상64GB
Memory (RAM)8GB4GB
ProcessorsCPU 코어의 절반2코어
Network AdapterNAT (기본값)-
⚠️ RAM 4GB 미만이면 Windows 11이 매우 느립니다. 최소 4GB, 권장 8GB!
6

Windows 11 설치

① 가상머신 시작

  1. 생성된 가상머신 선택 → [Play virtual machine]
  2. "Press any key to boot from CD..." 메시지 뜨면 즉시 아무 키 입력!
⚠️ 키 입력 타이밍을 놓치면 다시 재시작해야 합니다. 부팅 화면 뜨자마자 클릭!

② 설치 단계

  1. 언어: 한국어 → [다음] → [지금 설치]
  2. 제품 키: "제품 키가 없음" 클릭 (나중에 정품 인증 가능)
  3. 버전: Windows 11 Pro 권장 → [다음]
  4. 사용 조건 동의 → [다음]
  5. 설치 유형: "사용자 지정: Windows만 설치"
  6. 파티션 선택 → [다음]
✅ 설치 시간 약 15~30분. 재시작 여러 번 반복되는 것은 정상입니다!

VMware Tools 설치 (필수!)

설치 완료 후 VMware Tools를 반드시 설치해야 화면 해상도·복사/붙여넣기·파일 공유가 정상 작동합니다.

  1. VMware 상단 메뉴 → [VM] → [Install VMware Tools]
  2. 가상머신 탐색기에서 DVD 드라이브 열기
  3. setup64.exe 실행 → [Next] → [Install] → [Finish]
  4. 재시작
⚠️ VMware Tools 없으면: 화면 작음, 복붙 안됨, 파일 공유 불가 → 반드시 설치!

❓ 자주 묻는 질문

Q. 네이버 이메일로 가입이 안 돼요
@naver.com, @daum.net, @nate.com 등 국내 포털 이메일은 Broadcom에서 차단되어 있습니다. 반드시 Gmail(@gmail.com)이나 회사 이메일로 가입하세요.
Q. 인증 코드 이메일이 안 와요
스팸 폴더를 확인하세요. 제목은 "Broadcom Registration Notice - Verification Code"입니다. 5~10분 기다려도 안 오면 가입을 다시 시도하세요.
Q. "다운로드하려면 추가 인증이 필요합니다" 메시지가 나와요
정상입니다. 주소 입력 화면에서 대략적인 주소와 정확한 우편번호만 입력하면 됩니다. [I Agree] 동의 후 다시 다운로드 버튼 클릭하면 됩니다.
Q. 설치 후 "Not Licensed"라고 나와요. 오류인가요?
정상입니다! 2024년 11월부터 VMware는 라이선스 없이 완전 무료입니다. "Type: Personal Use Only", "Status: Not Licensed" 상태로 그냥 사용하시면 됩니다.
Q. 로그인했는데 계속 로그인 페이지로 돌아와요
브라우저 쿠키 문제입니다. ① 시크릿 모드 해제 ② 쿠키 허용 확인 ③ Chrome 또는 Edge 사용 권장 (Safari 차단될 수 있음)
Q. Windows 11이 TPM 오류로 설치 안 돼요
가상머신 설정 → [VM] → [Settings] → [Options] → [Advanced] → UEFI 선택 + Secure Boot 체크하세요.
Q. 가상머신이 너무 느려요
RAM을 8GB 이상으로 늘리고, VMware Tools가 설치됐는지 확인하세요. Tools 없으면 무조건 느립니다.
Q. 호스트 PC와 파일 공유하고 싶어요
VMware Tools 설치 후 → [VM] → [Settings] → [Shared Folders] → 공유할 폴더 추가하면 됩니다.
Q. 복사/붙여넣기가 안 돼요
VMware Tools가 설치되어 있어야 합니다. 설치 후 재시작하면 바로 됩니다.
