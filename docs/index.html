⚡ 주요 기능
🔗

URL/Git 자동 분석

링크를 보내면 AI가 분석해서 Guides/ 폴더에 가이드 문서를 자동 생성합니다.

TIL 자동 생성

오늘 배운 내용을 요청하면 TIL/YYYY-MM-DD 형식으로 자동 저장됩니다.

🗺️

MOC 자동 연결

저장된 노트가 주제별 지식 지도(MOC)에 자동으로 링크됩니다.

🧠

AI 장기 기억

MEMORY.md에 중요한 내용을 기록해 다음 세션에도 컨텍스트가 유지됩니다.

⚙️

완전 커스터마이징

HEARTBEAT.md 수정만으로 저장 규칙, 폴더 경로, 노트 스타일을 자유롭게 변경합니다.

🔔

텔레그램 브리핑

저장 완료 시 텔레그램으로 "📒 Obsidian 저장: 경로" 브리핑을 즉시 받습니다.

🚀 설치 방법

⏱️ 5분이면 완료됩니다. Node.js, Obsidian, Telegram만 미리 설치되어 있으면 됩니다.

1

텔레그램 봇 만들기

텔레그램에서 @BotFather 검색 → /newbot → 봇 이름 입력 → 토큰 복사

# 텔레그램 앱에서
@BotFather → /newbot
→ 봇 이름 입력 (예: 나의AI)
→ username 입력 (예: myai_bot)
→ 토큰 복사: 1234567890:ABCdef...
2

레포 클론 & 설치 실행

git clone 하나로 모든 파일이 자동으로 설정됩니다.

# Windows
git clone https://github.com/aiebrain/openclaw-obsidian-starter.git
cd openclaw-obsidian-starter
.\setup.ps1

# Mac / Linux
git clone https://github.com/aiebrain/openclaw-obsidian-starter.git
cd openclaw-obsidian-starter && ./setup.sh
3

설치 스크립트 답변

스크립트가 이름, Obsidian 경로, 텔레그램 토큰을 물어봅니다. 입력하면 자동 설정됩니다.

이름: 만성
Obsidian 볼트 경로: C:\Users\나\obsidian
텔레그램 봇 토큰: 1234567890:ABCdef...
4

OpenClaw 실행 & 대화 시작

Obsidian 앱에서 볼트를 열고, OpenClaw를 실행합니다.

openclaw gateway start
# 텔레그램에서 봇에게 메시지 보내기 시작! 🎉

AI가 자동으로 기록하는 이유는 HEARTBEAT.md 파일 덕분입니다.
이 파일에 저장 규칙을 쓰면, AI가 대화 중 그 규칙대로 자동 판단하고 저장합니다.

## Obsidian 자동 저장 규칙

볼트 경로: C:\Users\나\obsidian

# 어떤 내용을 어디에 저장할지 AI가 이 규칙을 따릅니다
- 가이드성 내용 → Guides/
- 오늘 배운 것  → TIL/YYYY-MM-DD 제목.md
- URL 공유 시   → 자동 분석 후 Guides/ 저장
- 저장 후 브리핑: "📒 Obsidian 저장: 경로/파일명"
