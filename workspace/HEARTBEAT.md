## 📂 Obsidian 자동 저장 규칙

### 볼트 경로 (setup.ps1 실행 시 자동 설정됨)
볼트 루트: {{OBSIDIAN_VAULT_PATH}}

### 자동 저장 트리거
- 대화 중 기억할 만한 내용 → 별도 요청 없이 즉시 Obsidian 기록
- 기록 후 브리핑만 제공: "📒 Obsidian 저장: `경로/파일명`"
- URL 또는 Git 링크 공유 시 → 자동 분석 후 Guides/ 에 가이드 생성

### 폴더별 저장 규칙
- 가이드성 내용 → `Guides/`
- 오늘 배운 것 → `TIL/YYYY-MM-DD 제목.md`
- 프로젝트 관련 → `Projects/`
- 빠른 메모/캡처 → `Inbox/` (나중에 정리)
- 새 주제 → 해당 MOC에 링크 추가

### 노트 작성 스타일
- 제목에 이모지 사용: `## 🎥 제목`
- 번호 매긴 섹션 구조: `### 1. 개요`, `### 2. 사용법`
- 핵심 정보는 **표(table)** 로 정리
- 코드/설정은 **코드블록** 사용 (언어 태그 포함)
- FAQ 섹션 포함 (Q: / A: 형식)
- 태그 포함: `#주제` 형식
- Obsidian 내부 링크: `[[노트 이름]]`
- 문서 끝에 관련 노트 링크 연결

### MOC 연동 규칙
- AI 도구 관련 → `MOC/AI 도구 모음.md`에 링크 추가
- 개발 도구 관련 → `MOC/개발 도구.md`에 링크 추가
- 프로젝트 관련 → `MOC/프로젝트 허브맵.md`에 링크 추가
- 콘텐츠 관련 → `MOC/콘텐츠 & 블로그 맵.md`에 링크 추가

## 시스템 상태 모니터링
- 30분마다 stale-socket 재연결은 정상 동작 → 알림 불필요
- ERROR 로그 연속 5회 이상 → 사용자에게 알림
- 크론잡 에러 발생 시 → 즉시 수정 후 텔레그램 브리핑
